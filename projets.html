Projets Universitaires

Projet Deep Learning - Détection et Reconnaissance faciale

Développement d'un système de détection et de reconnaissance faciale en temps réel utilisant le modèle YuNet et la bibliothèque OpenCV. Il permet d’identifier des visages à partir d’images ou de vidéos en comparant des embeddings faciaux préenregistrés.

Mediapipe Yunet OpenCv

Projet Deep Learning - Classification MNIST

Développement d'un modèle de deep learning en PyTorch, comprenant un réseau de neurones profond et un CNN pour la classification des images manuscrites du dataset MNIST.

Pytorch Multi-Layer Perceptron (MLP) , Convolutional Neural Networks (CNN)

Projet IA - Jeu de puissance 4

Création d’un jeu de puissance 4 en java et implémentation de différents algorithmes (Minimax, Alpha-Beta, MCTS).

Java Minimax Alpha-Beta MCTS

Projet IA - Carte Kohonen

Développement d’une carte auto-organisatrice de Kohonen pour la prédiction des positions d’un bras robotique à partir de données motrices.

Python Numpy Matplotlib

Système de contrôle d'accès à Polytech

Conception d'un système de gestion d'accès sécurisé basé sur la technologie NFC et un microcontrôleur ESP32. Le projet inclut le développement d'une application mobile permettant de superviser les accès en temps réel, de gérer les utilisateurs et d'assurer la synchronisation des données via Firebase.

Kotlin Arduino C++ Firebase
